# Changelog

All notable changes to TrueMeter are documented here.

## [0.2.1] - 2026-07-13

- Added the live demo, release automation, npm provenance publishing, and repository contribution guidance.

## [0.2.0] - 2026-07-13

- Published package metadata for npm distribution.
- Added a per-display calibration storage safeguard for identical configured storage keys.
- Added a regression test for calibration persistence.
- Updated installation documentation for npm.

## [0.1.1] - 2026-07-13

- Added production package exports, build validation, and CI checks.
- Expanded documentation for display confidence, privacy, calibration, and source data.

## [0.1.0] - 2026-07-13

- Initial release of the browser display calibration resolver.
- Added ScreenRes, EDID, device-profile, Client Hints, and Apple internal-display sources.
- Added local per-display calibration and browser zoom-aware scale calculation.
